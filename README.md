# Vida Immobilien — Immersive Site

Next.js-Projekt mit scroll-gesteuertem Video-Erlebnis.

## Entwicklung

```bash
cd immersive-site
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000).

## Struktur

- `immersive-site/` — Next.js App (App Router)
- `index.html` — Legacy/Alternativ-Datei im Projektroot
